# Task 1: Data Cleaning - Mall Customer Segmentation Dataset

## Dataset Used
Mall_Customers.csv (Kaggle)

## Steps Performed to clean the dataset as follows
1. Loaded the dataset using Pandas.
2. Checked for missing values.
3. Removed duplicate rows.
4. Standardized text in Gender column.
5. Cleaned and reformatted column names.
6. Verified and corrected data types.
7. Saved the cleaned dataset as cleaned_mall_customers.csv.

## Files Included
- Mall_Customers.csv (raw data)
- cleaned_mall_customers.csv (cleaned version)
- Readme file (this one)

## Tools Used
- Python
- Pandas
- Jupyter Notebook / VS Code

## Summary
This project involved preprocessing a Kaggle dataset, fixing formatting issues, and preparing it for analysis.
